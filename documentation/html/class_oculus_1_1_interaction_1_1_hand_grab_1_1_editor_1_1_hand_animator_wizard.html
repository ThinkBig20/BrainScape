Detailed Description

This wizard helps creating a set of fixed Animation Clips using HandTracking to be used in a skinned synthetic hand with an Animator. Assign a HandSkeletonVisual and click the buttons as you perform the relevant poses with your tracked hand. The output will be an Animator that can be directly used in a Skinned hand. Once you are done you can automatically create the opposite hand data by providing the strings internally used for differentiating the left and right transforms. (typically l and r) Works great in conjunction with FromOVRControllerHandData.cs.
