Detailed Description

The DistanceHandGrabInteractable allows grabbing the marked object from far away. Internally it uses HandGrabPoses to specify not only the poses of the hands but the required gestures to perform the grab. It is possible (and recommended) to reuse the same HandGrabPoses used by the HandGrabInteractable, and even select just a few so they become the default poses when distant grabbing.
