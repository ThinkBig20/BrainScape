Detailed Description

A helper class that keeps track of the current state of features, quantized into corresponding FeatureStates.

Template Parameters
TFeatureAn enum containing all features that can be tracked.
TFeatureStateAn enum of all the possible states of each member of the TFeature param. The name of each member of this enum must be prefixed with one of the values of TFeature.
Type Constraints
TFeature :unmanaged
