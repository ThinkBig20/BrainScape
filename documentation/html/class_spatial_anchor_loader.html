Detailed Description

Demonstrates loading existing spatial anchors from storage.

Loading existing anchors involves two asynchronous methods:

  1. Call OVRSpatialAnchor.LoadUnboundAnchors
  2. For each unbound anchor you wish to localize, invoke OVRSpatialAnchor.UnboundAnchor.Localize.
  3. Once localized, your callback will receive an OVRSpatialAnchor.UnboundAnchor. Instantiate an OVRSpatialAnchor component and bind it to the UnboundAnchor by calling OVRSpatialAnchor.UnboundAnchor.BindTo.
