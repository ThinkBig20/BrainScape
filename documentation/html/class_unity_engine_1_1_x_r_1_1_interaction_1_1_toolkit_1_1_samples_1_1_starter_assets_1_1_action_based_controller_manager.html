const int UnityEngine.XR.Interaction.Toolkit.Samples.StarterAssets.ActionBasedControllerManager.k_UpdateOrder = XRInteractionUpdateOrder.k_Controllers - 1
static

Order when instances of type ActionBasedControllerManager are updated.

Executes before controller components to ensure input processors can be attached to input actions and/or bindings before the controller component reads the current values of the input actions.
