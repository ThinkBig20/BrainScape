Detailed Description

OVR Component to drive blend shapes on a SkinnedMeshRenderer based on Face Tracking provided by OVRFaceExpressions.

See OVRFace for more information. This specialization of OVRFace provides mapping based on an array, configurable from the editor This component comes with a custom editor that supports attempting to auto populate the mapping array based on string matching See OVRCustomFaceEditor for more information.
