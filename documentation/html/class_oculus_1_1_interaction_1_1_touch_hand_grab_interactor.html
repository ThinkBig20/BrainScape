Detailed Description

TouchHandGrabInteractor provides a hand-specific grab interaction model where selection begins when finger tips overlap with an associated interactable. Upon selection, the distance between the fingers and thumb is cached and is used for determining the point of release: when fingers are outside of the cached distance.
