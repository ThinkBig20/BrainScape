◆ ScaleChildren

bool OVRSceneVolume.ScaleChildren
getset

Whether the child transforms will be scaled according to the dimensions of this volume.

If set to True, all the child transforms will be scaled to the dimensions of this volume immediately. And, if it's set to False, dimensions of this volume will no longer affect the child transforms, and child transforms will retain their current scale.
